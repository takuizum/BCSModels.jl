BCSModels.jl

Bayesian Covariance Structure Modeling for Item Response Theory in Julia.

BCSModels.jl implements the Bayesian Covariance Structure Model (BCSM) family of Fox, Mulder, Klotzke and collaborators with two posterior inference engines:

  • the closed-form Gibbs sampler from the BCSM literature (Klotzke & Fox 2019; Fox, Wenzel & Klotzke 2021; Fox 2024), and
  • an original closed-form mean-field coordinate-ascent variational Bayes (CAVI) routine — to our knowledge the first VB inference engine for the BCSM family.

The two engines share the same priors and Albert–Chib augmentation, so they can be compared head-to-head on identical simulated and real data.

30-second tour

using BCSModels, Random, Statistics

Y, info = simulate_irt_bcsm(MersenneTwister(1), 500, 10; θ_true=0.4)
model = IRTBCSM(K=10)

gibbs = gibbs_irt_bcsm(Y, model; niter=1500, burnin=500)
vb    = cavi_irt_bcsm(Y, model; maxiter=400, tol=1e-7)

mean(gibbs.samples_θ)   # ≈ 0.4    (truth)
vb.m_θ[1]               # variational mean of θ

For the multi-testlet model:

testlet_of = repeat(1:3, inner=4)   # 12 items, 3 testlets of size 4
Y, info = simulate_testlet_bcsm(MersenneTwister(2), 600, 12;
                                 testlet_of=testlet_of,
                                 θ_true=[0.3, 0.5, 0.2])
model = TestletBCSM(K=12, testlet_of=testlet_of)
res = gibbs_testlet_bcsm(Y, model; niter=1500, burnin=500)
mean(res.samples_θ, dims=1)   # per-testlet posterior means

The BCSM model in one paragraph

A hierarchical IRT model with random person or testlet effects induces a marginal covariance among the latent response utilities once the random effects are integrated out. BCSM places the prior directly on that marginal covariance:

\[\boldsymbol{\Sigma} \;=\; \boldsymbol{\Sigma}_0 \;+\; \sum_{t=1}^T \theta_t \, \mathbf{u}_t \mathbf{u}_t^{\!\top}, \qquad \theta_t \in \mathbb{R}.\]

Each $\mathbf{u}_t$ is a known binary item-grouping indicator (testlet membership, group, response type) and each $\theta_t$ is a covariance component that, unlike a variance, is allowed to be negative. With a truncated-shifted-inverse-gamma prior on each $\theta_t$, the conditional posteriors are all closed form, enabling both a fast Gibbs sampler and the mean-field CAVI that this package contributes.
